# Frontend (Vite + React)

## Daily development — **no build needed**

```bash
npm run dev
```

Open the URL Vite prints (usually `http://localhost:5173`). **Save your files** — the app updates in the browser automatically (HMR). You do **not** need `npm run build` after each change.

## When to run build

- **Deploying** to production (serves the `dist/` folder)
- **CI** or checking that a production bundle compiles

```bash
npm run build
```

## Lint (optional)

```bash
npm run lint
```
